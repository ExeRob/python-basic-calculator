--Python Basic Calculator--

A simple console-based calculator written in Python.
This project performs basic arithmetic operations using functions and includes error handling for invalid input.

Features:
Addition
Subtraction
Multiplication
Division (with zero-division protection)
Input validation
Clean and simple structure

How It Works:
The calculator runs in a loop and lets the user choose an operation.
It asks for two numbers and performs the selected calculation using separate functions.

Running the Program:
Make sure you have Python installed.
Run the script from your terminal:
python basic_calculator.py
